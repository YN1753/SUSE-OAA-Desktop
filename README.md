# 🎓 桌面课程表 & 系统监控助手 (Desktop Schedule & Monitor)

一款基于 **Qt (C++)** 开发的高颜值桌面悬浮插件。集成了教务课表同步、实时系统硬件监控、灵动岛模式以及动态待办事项管理功能。支持多主题切换，旨在打造极致的桌面生产力工具。

---

## 🌟 核心功能

### 1. 三位一体布局模式
通过右键菜单可精准切换三种显示形态，适配不同办公场景：
* **经典竖版 (Expanded)**：详细展示今日/明日课表、倒计时及系统摘要。
* **微型灵动岛 (Pill)**：极致简约，仅在桌面顶部显示当前时间与下一节课提醒。
* **控制中心大屏 (Dashboard)**：横向三栏布局，集成详细时钟、完整课表、待办清单及图形化硬件监控。

### 2. 智能课表与倒计时
* **多状态展示**：自动识别“进行中”、“距离上课”、“今日无课”等状态，并动态调整显示颜色。
* **自动时钟化**：当无课或处于自由时间时，倒计时区域自动转化为高对比度的数字时钟。
* **教务同步**：内置 Python 爬虫接口，支持从学校教务系统抓取最新课表。

### 3. 高颜值登录与配置系统
* **自定义 UI**：采用深色圆角卡片式登录界面，告别系统原生对话框。
* **静默刷新**：支持账号密码本地 JSON 加密存储，实现右键“快速刷新”免输入同步。

### 4. 动态待办事项 (To-Do List)
* **交互友好**：支持回车快速添加、点击勾选状态、右键删除任务。
* **持久化存储**：任务状态实时保存至 `todos.json`，重启不丢失。

## 🎨 主题风格

支持三种精心设计的配色方案，满足不同审美需求：
* **现代极简 (Minimalist)**：白灰色系，适配浅色壁纸，清新自然。
* **赛博朋克 (Cyberpunk)**：深黑底色配霓虹粉青，极具科技感。
* **清新实用 (Clean)**：高对比橙绿配色，经典耐看，指引性强。

## 🛠️ 技术亮点

### 1. 避让机制：解决中文输入冲突
针对中文拼音输入被打断的痛点，项目采用了 **“分钟级刷新”** 与 **“焦点避让”** 技术：
* **按需重排**：只有在分钟跳转或风格切换时才清空并重构布局，避免了每秒刷新导致输入法窗口关闭的问题。
* **精准更新**：硬件监控与时间秒数通过直接修改属性更新，不触发布局重绘，确保输入过程丝滑。

### 2. 模块化架构
* **UIManager**: 负责 UI 构建、样式表(QSS)应用及刷新逻辑。
* **DataManager**: 负责硬件信息采集与 JSON 数据解析。
* **MenuManager**: 负责多级右键菜单逻辑。
* **MainWindow**: 负责窗口管理、无边框拖拽实现及信号调度。

## 🚀 快速开始

### 开发环境
* **Qt SDK**: 5.12+ (或 Qt 6)
* **标准**: C++ 11
* **编译器**: MSVC 或 MinGW

### 运行说明
1. 确保项目目录下存在编译后的 Python 同步程序 `main.exe`。
2. 首次运行建议通过“教务登录”功能同步个人课表。
3. 课表数据将保存在 `schedule.json`，待办事项保存在 `todos.json`。
